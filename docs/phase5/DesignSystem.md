# BARQ Fleet Management - Design System

**Created:** December 6, 2025
**Phase:** 5 - Visual Design & Prototyping
**Version:** 1.0

---

## Design Principles

1. **Clarity First** - Information should be immediately understandable
2. **Efficient Workflows** - Minimize clicks for common tasks
3. **Consistent Patterns** - Same interaction = same behavior everywhere
4. **Accessible by Default** - WCAG 2.1 AA compliant
5. **Mobile-Ready** - Works across all device sizes

### Animations

```css
/* Fade In */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* Slide Up */
@keyframes slideUp {
  from { transform: translateY(10px); opacity: 0; }
  to { transform: translateY(0); opacity: 1; }
}

/* Scale In */
@keyframes scaleIn {
  from { transform: scale(0.95); opacity: 0; }
  to { transform: scale(1); opacity: 1; }
}
```
