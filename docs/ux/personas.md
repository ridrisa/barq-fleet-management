## Persona 1: Fleet Operations Manager

### Profile
- **Name:** Ahmed Al-Rashid
- **Age:** 38
- **Role:** Fleet Operations Manager
- **Experience:** 12 years in logistics
- **Tech Savviness:** Intermediate (Excel expert, learns new systems quickly)
- **Work Environment:** Office-based, occasional site visits

### Goals & Motivations
- Maximize fleet utilization (target: 85%+)
- Reduce vehicle downtime through preventive maintenance
- Ensure all vehicles meet compliance requirements
- Control fuel costs and detect anomalies
- Track courier performance and productivity

### Pain Points
- Document tracking for 200+ couriers is chaotic
- Maintenance is reactive, not preventive
- Multiple systems for fleet, HR, and FMS data
- No unified view of vehicle health
- Manual Excel reports take hours

### Key Tasks
| Task | Frequency | Time Spent |
|------|-----------|------------|
| Review vehicle status dashboard | Daily | 30 min |
| Approve maintenance requests | Daily | 45 min |
| Check document expiration alerts | Weekly | 1 hour |
| Analyze fuel consumption | Weekly | 2 hours |
| Generate fleet reports | Weekly | 3 hours |
| Conduct fleet audits | Monthly | 8 hours |

### Success Metrics
- Fleet utilization rate (target: 85%)
- Vehicle availability (target: 95%)
- Maintenance compliance (target: 100%)
- Fuel cost per km (target: < 0.5 SAR)
- Document compliance rate (target: 100%)

### Feature Priorities
1. Real-time vehicle tracking (FMS integration)
2. Automated maintenance scheduling
3. Document expiration alerts
4. Fuel analytics dashboard
5. Courier performance metrics

## Design Implications

### Navigation
- Fleet Manager: Quick access to dashboards and alerts
- HR Admin: Streamlined payroll and leave workflows
- Dispatch: Real-time focused, minimal clicks
- Admin: Power user features, advanced search
- Support: Unified search, quick actions

### Dashboard Customization
- Role-based default dashboards
- Configurable widgets per persona
- Saved views and filters
- Keyboard shortcuts for power users

### Mobile Needs
- Dispatch: Full-featured mobile app (critical)
- Support: Mobile ticket handling
- Fleet Manager: Mobile alerts and approvals
- HR Admin: Leave approval notifications
